## 🔍 Data Mining Pipeline

### 1️⃣ Data Understanding & Exploration
- Loaded and inspected dataset structure, shape, and data types
- Identified missing values, duplicates, and outliers
- Analysed feature distributions and class balance
- Visualised relationships using correlation heatmaps and pair plots

### 2️⃣ Data Preprocessing
- Handled missing values (imputation / removal)
- Encoded categorical variables (Label Encoding / One-Hot Encoding)
- Normalised and standardised numerical features
- Removed irrelevant or redundant features

### 3️⃣ Data Mining Techniques Applied

#### 🔷 Classification
- Trained supervised models to predict target categories
- Models used: Decision Tree, Random Forest, Naïve Bayes, KNN
- Evaluated using Accuracy, Precision, Recall, F1-Score, Confusion Matrix

#### 🔷 Clustering *(Unsupervised Learning)*
- Applied **K-Means Clustering** to group similar data points
- Used **Elbow Method** and **Silhouette Score** to determine optimal k
- Visualised clusters using 2D scatter plots

#### 🔷 Association Rule Mining
- Applied **Apriori Algorithm** to discover frequent itemsets
- Generated association rules with Support, Confidence, and Lift metrics
- Identified key patterns and relationships within the dataset

### 4️⃣ Evaluation & Interpretation
- Compared model performance across multiple metrics
- Visualised results: confusion matrices, ROC curves, cluster plots
- Interpreted discovered patterns and rules in the context of the dataset
